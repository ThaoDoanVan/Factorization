Require installation of python3 library tkinter
To start the GUI, run: python3 projectGUI.py
Choose the method and load your input file with correct format
For example, if you want to factor 27669672886940111406731995004890116126494227271653 using the three method with the following parameters:
p_max = 10^6, B = 10^6, B1 = 10^7, B2 = 10^8
then the input file should be as below:

n = 27669672886940111406731995004890116126494227271653
p_max = 1000000
B = 1000000
B1 = 10000000
B2 = 100000000

The GUI displays the sample input format for each method
